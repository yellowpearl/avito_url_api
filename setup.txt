1. Нужно сделать git clone репозитория
2. Зайти в папку avito_url_shortner_api
3. В терминале прописать: pip install -r requirements.txt
4. Создать бд: python3 manage.py migrate
5. Создать пользователя: python3 manage.py createsuperuser
6. Запустить сервер: python3 manage.py runserver
После этого можно добавлять новые записи в базу данныех, есть несколько способов:
	1. Через админку http://127.0.0.1:8000/admin/shortener/link/add/ - Следить чтобы у ссулок был https:// префикс
	2. В адресно строке вбить data:text/html,<form action=http://127.0.0.1:8000/s/short method=post><input name=a></form>
	Появится поле для ввода, в него можно вписывать нужные ссылки, если отправить форму, то придет ответ с хешом ссылки
	3. POST запрос с помощью postman
	4. В файле avito_url_api/shortener/tests.py есть несколько примеров с модулем requests, можно запустить скрипт и в терминале выведется хеши ссылок, по которым можно перейти
В ответ на пост запросы программа вернет хеш, по нему можно перейти по ссылке http://127.0.0.1:8000/s/"hash",
Если хеш есть в базе данных, то вас перекинет на полную ссылку
